Welcome to the 5th Python meetup in Denmark. This is a free-for-all event which aim is to bring the Python community in Denmark together.

If you want to receive emails about future meetups, write to meetup@pycon.dk.

Everyone is free to give a talk. Just send us the title of yout talk to meetup@pycon.dk.

If you want to discuss something publicly, we have a sub-reddit for that.
