When: 18:00, Thursday, 4th April 2013

Where: Room M1, Diplomvej 381, Scion DTU, 2800 Lyngby

Welcome to the 3rd Python meetup in Denmark. This is a free-for-all event which aim is to bring the Python community in Denmark together.

If you want to come or give a talk, write us an email to meetup@pycon.dk.

We have 2 formats for talks:

  • 30-minute full-blown talks
  • 5-minute "lightning" talks

If you want to talk to a human being call (+45) 53 42 53 21.

If you want to discuss something publicly, we have a sub-reddit for that.

Schedule

  • 18:00: Python for beginners (by Jóan Petur Petersen)
  • 19:00: Raspberry Pi and Python (by Sergejus Martinenas)
  • 19:30: How to write an interpreter (by Vladimir Keleshev)

The first talk is mostly targeting people that are new to Python. So if you know Python well you can come at 19:00 to hear just the more advanced talks.
